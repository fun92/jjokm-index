# 쪼꼼 인덱스 1.2.0

맥 화면 오른쪽에 작은 하트로 숨어 있다가 클릭하면 열리는 사이드 메모 앱입니다.

## Highlights

- 작은 `💛` 버튼으로 접힘
- 오른쪽 가장자리 근처에서 하트가 선명하게 나타나는 hover reveal
- 메뉴바의 `💛`로 열기/접기/종료
- 메뉴바에서 로그인 시 자동 열기 설정
- 메뉴바에서 왼쪽/오른쪽 위치 전환
- 메뉴바에서 메모 고정 설정
- 하트 위치 드래그 조정
- 여러 메모 탭
- 자동 저장
- 굵게/기울임/밑줄/글머리 서식
- macOS 색상 선택창으로 글자 색 지정
- `Aa` 슬라이더로 글자 크기 조정
- 로그인 시 자동 열기 설치/해제 스크립트 포함

## 설치

1. `jjokkom-index-1.2.0.zip`을 다운로드합니다.
2. 압축을 풉니다.
3. `쪼꼼 인덱스.app`을 Applications 폴더로 옮깁니다.
4. 처음 실행할 때 macOS 경고가 보이면 앱을 우클릭한 뒤 `열기`를 선택합니다.

압축을 iCloud Drive에서 풀었다면 앱을 그 자리에서 실행하지 말고 Applications 폴더로 옮긴 뒤 실행하세요.

## 알려진 제한

- 현재 배포 파일은 Apple Developer ID 공증 전입니다.
- 굵게/색상 같은 리치 텍스트 서식 저장은 추후 개선 대상입니다.
